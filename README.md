# IoT: Entre Dados e Destino

**IoT: Entre Dados e Destino** é um jogo narrativo educativo sobre Internet das Coisas, privacidade e segurança digital.

O projeto busca apresentar, de forma interativa, situações comuns envolvendo dispositivos conectados, coleta de dados, escolhas digitais e consequências relacionadas à exposição de informações pessoais.

## Sobre o jogo

O jogador acompanha uma narrativa em estilo visual novel/arcade retrô, tomando decisões que influenciam o desenvolvimento da história.

A proposta é incentivar a reflexão sobre temas como:

- Privacidade digital;
- Segurança em dispositivos IoT;
- Coleta e uso de dados pessoais;
- Riscos de exposição de informações;
- Escolhas conscientes no uso da tecnologia.

## Tecnologias utilizadas

O projeto foi desenvolvido em um único arquivo HTML, com:

- HTML;
- CSS;
- JavaScript;
- Interface responsiva para uso em computador e tablet;
- Estética retrô/pixel art com elementos de visual novel.

## Como executar

Para jogar localmente:

1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` em um navegador moderno.
3. Recomenda-se usar Google Chrome, Microsoft Edge ou Firefox.

Também é possível acessar o projeto pelo GitHub Pages.
